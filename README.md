📝 Descripción
Desarrolla una página web usando HTML y CSS que permita a los usuarios suscribirse a un Newsletter. Este reto te ayudará a practicar tus conocimientos básicos de diseño web, incluyendo el uso de campos de entrada (input) y botones.




🎯 Objetivos
Comprender la sintaxis básica de HTML y CSS.
Estructurar y dar estilo a una página web.
Crear y dar formato a contenido textual.
Trabajar con campos de entrada (input) y botones.
Agregar imágenes o íconos representativos.



🧩 Requerimientos
Incluir un ícono que indique la finalidad de la página.
Incluir un campo de entrada para que los usuarios escriban su correo electrónico.
Incluir un botón para enviar el formulario.
Mantener un diseño limpio, con buen espaciado y tipografía adecuada.
Aplicar sombra (box-shadow) en los elementos:
box-shadow: 0px 20px 30px 0px rgba(0, 0, 0, 0.03),
            0px 8px 12px 0px rgba(0, 0, 0, 0.08);


            
        
⚙️ Detalles técnicos
Usar únicamente HTML y CSS.
Puedes emplear cualquier editor de texto o IDE.
